A10 - Misleading Visualizations

This assignment explores how data visualizations can be manipulated to support false narratives, and how to create ethical alternatives that represent data honestly.

Dataset: gaming-data.json

Description: This dataset represents a fictional student's weekly hours spent playing video games and their corresponding exam scores over a 7-week period. The data contains 7 data points with three fields: week number, hours spent gaming, and exam score.

Misleading Claim: "Playing more video games directly improves academic performance."

D3.js Visualizations

Deceptive Chart - Manipulated to Support False Claim Ethical Chart - Honest Representation of Same Data Side-by-Side Comparison

Manipulation Techniques Used

  1. Truncated Y-Axis: Starting the y-axis at 75 instead of 0 makes a 15-point improvement appear as a dramatic near-vertical climb.
  2. Exaggerated Aspect Ratio: Using a tall, narrow chart (high height-to-width ratio) visually steepens the slope, making the trend appear more significant.
