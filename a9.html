A9: Geospatial Data Visualization

This assignment explores geospatial visualization using D3.js, creating a proportional symbols map to visualize US state cost of living data.

Visualization

📍 Proportional Symbols Map: US Cost of Living
Bubble size represents cost of living index, color represents housing cost. Hover for details.

Data Sources

  • Cost of Living Data: expensive-states.csv - Contains costRank, State, costIndex, groceryCost, housingCost, utilitiesCost, transportationCost, and miscCost
  • State Centroids: state-centroids.csv - Latitude and longitude coordinates for each US state centroid

Features

  • Proportional symbols (circles) sized by Cost of Living Index
  • Color encoding for Housing Cost using sequential red scale
  • Interactive tooltips showing state details on hover
  • Size legend showing cost index values
  • Color legend showing housing cost scale
